Селектор Описание Пример
* Универсальный селектор, применится к каждому элементу на странице * {...}
X Селектор по тегу, применится ко всем тегам X a {...}
.X Селектор по классу, применится ко всем элементам с классом X .classname {...}
#X Селектор по id, применится ко всем элементам с id X #someId {...}
[X="attr"] Селектор по атрибуту, применится к элементам, у которых значение атрибута X = attr [target="_blank"] {...}
#X.Y Комбинированный селектор - для элементов, к которым применятся все перечисленные селекторы a.classname[target="_blank"] {...}
X Y Контекстный селектор - для элементов У, вложенных в X .wrapper p {...}
X > Y Селектор дочерних элементов - выберет только непосредственных потомков .wrapper > p {...}
X + Y Выберет ближайшего и соседа элемента X .element + .sibling {...}
X ~ Y Выберет всех соседей элемента X .element ~ .sibling {...}
